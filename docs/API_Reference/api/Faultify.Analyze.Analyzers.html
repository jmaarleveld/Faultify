Namespace Faultify.Analyze.Analyzers

Classes

ArithmeticAnalyzer

Analyzer that searches for possible arithmetic mutations inside a method definition. Mutations such as '+' to '-', '*', '/', and '%'.

ArrayAnalyzer

Analyzer that searches for possible array mutations inside a method definition. Limitations:

  • Only one-dimensional arrays.
  • Only arrays that are created dynamically with more than 2 values which are not default values.
  • Only array of the following types: double, float, long, ulong, int, uint, byte, sbyte, short, ushort, char, boolean.

BitwiseAnalyzer

Analyzer that searches for possible bitwise mutations inside a method definition. Mutations such as such as 'and' and 'xor'.

BranchingAnalyzer

Analyzer that searches for possible boolean branching mutations inside a method definition. Mutations such as 'if(condition)' to 'if(!condition)'.

ComparisonAnalyzer

ConstantAnalyzer

Analyzer that searches for possible constant mutations inside a type definition, this class is the parent class to all constant analyzers.

LogicalAnalyzer

OpCodeAnalyzer

Analyzer that searches for possible opcode mutations inside a method definition. A list with opcodes definitions can be found here: https://en.wikipedia.org/wiki/List_of_CIL_instructions

VariableAnalyzer

Analyzer that searches for possible variable mutations. Mutations such as 'true' to 'false'
