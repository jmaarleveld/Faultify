Namespace Faultify.Analyze.AssemblyMutator

Classes

AssemblyMutator

The AssemblyMutator can be used to analyze all kinds of mutations in a target assembly. It can be extended with custom analyzers. Though an extension must correspond to one of the following collections in AssemblyMutator:

  • ArrayMutationAnalyzers(ArrayAnalyzer)
  • ConstantAnalyzers(VariableAnalyzer)
  • VariableMutationAnalyzer(ConstantAnalyzer)
  • OpCodeMutationAnalyzer(OpCodeAnalyzer)


    If you add your analyzer to one of those collections then it will be used in the process of analyzing. Unfortunately, if your analyzer does not fit the interfaces, it can not be used with the AssemblyMutator.
